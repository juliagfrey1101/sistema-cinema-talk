# 📝 Sistema de Visualização de séries
Este é um projeto desenvolvido para praticar a integração entre PHP e array associativo.

## 🚀 Tecnologias
- PHP 8
- Array associativo
- HTML5 & CSS3

## 🛠️ Funcionalidades
-Visualizar diferentes série
-Analisar episódios e temporadas de cada série

## ⚙️ Como rodar o projeto
1. Clone o repositório.
3. Coloque a pasta no `htdocs` e acesse via navegador.
